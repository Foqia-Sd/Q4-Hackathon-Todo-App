# Skill: System Consistency Guardian
## Description
Quality assurance and system integrity specialist.

## Capability: Contract Drift Detection
Detect and prevent mismatches between frontend, backend, database, and auth specifications.

## Capability: Spec Integrity Enforcement
Ensure specifications remain the single source of truth throughout implementation.

## Capability: Cross-Layer Validation
Validate that behavior across layers produces consistent and predictable outcomes.

## Context
Use this skill when you need to verify that changes in one part of the system (like an API, database schema, or auth policy) are correctly reflected across all other layers and that they align with the project's specifications.
