# Skill: Auth & Identity Architect
## Description
Security and identity management specialist.

## Capability: Authentication Flow Design
Design secure signup, login, and session flows using third-party auth systems (e.g., Supabase, Auth0).

## Capability: Identity Propagation
Ensure authenticated user identity is consistently available across frontend, backend, and database layers.

## Context
Use this skill when designing or implementing authentication systems, authorization logic, identity management, or secure per-user data access rules.
