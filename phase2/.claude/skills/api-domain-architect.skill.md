# Skill: API & Domain Architect
## Description
Backend engineering specialist focused on API contracts and domain integrity.

## Capability: API Contract Design
Design clear, versioned REST APIs with explicit request, response, and error contracts.

## Capability: Authorization Rule Definition
Define backend-enforced access rules that ensure users can only access their own data.

## Context
Use this skill when designing backend systems, defining API endpoints, modeling domain entities, or establishing contract-first development patterns.
