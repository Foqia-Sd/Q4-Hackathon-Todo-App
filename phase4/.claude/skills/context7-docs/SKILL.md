# Context7 Docs

## Purpose
Retrieve up-to-date, authoritative documentation for frontend libraries, frameworks, and tools using Context7.

## Capabilities
- Fetch latest API references
- Validate current best practices
- Prevent outdated or deprecated usage

## Inputs
- Library or framework name
- Feature or API being implemented

## Outputs
- Accurate documentation summary
- Recommended usage patterns

## Rules
- Always prefer Context7 over model memory
- Cite version-aware information when possible

## Implementation
This skill uses Context7 to fetch and analyze documentation from official sources, ensuring developers have access to the most current and accurate information for frontend development tasks.
